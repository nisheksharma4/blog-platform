# Blog Platform

Full Stack Blog Application built using:

- Spring Boot
- MongoDB
- ReactJS
- Cloudinary

## Architecture
Modular Monolithic Architecture

## Features
- Create Blog
- Upload Images
- View Blogs
- Update & Delete

## Tech Stack
Backend: Spring Boot + MongoDB  
Frontend: React  
Deployment: Render + Vercel
